# symfony 1.4

## Установка
------------
1 Способ:
```
    git init # ваш проект
    git submodule add https://github.com/vadimDol/symfony1 vendor/symfony
    git submodule update --init --recursive
```

Скоро будет добавлена возможность установки через composer
